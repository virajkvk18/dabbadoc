# DabbaDoc UI MVP

A colorful, animated React/Vite frontend MVP for the DabbaDoc hackathon project.

## Run in VS Code

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal, usually:

```bash
http://localhost:5173
```

## Project Structure

```text
dabbadoc-ui/
├── index.html
├── package.json
├── README.md
└── src/
    ├── main.jsx
    ├── App.jsx
    └── styles.css
```

## Current MVP Mode

- All dashboard data is mock/static.
- Try Demo Analysis shows an AI processing modal.
- Upload Receipt opens a mock upload modal.
- DabbaBot chatbot has predefined responses.
- Backend/OCR/ML/GenAI can be connected later through API calls.
